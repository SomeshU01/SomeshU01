## 👋 Welcome — Read This First

You don't need a computer science degree.
You don't need years of IT experience.
You don't need expensive courses or enterprise software licenses.

**You need curiosity, consistency, and this repository.**

`Blue-Team-Mastery` is a free, structured, beginner-first curriculum for anyone who wants to build real cybersecurity skills — specifically the defensive, detection, and response skills that SOC (Security Operations Center) analysts use every single day.

Everything in this repo assumes you are starting from zero. Every concept is explained before it is used. Every tool has a setup guide. Every lab has expected outputs so you know when you've done it right.

This isn't a link dump. It's a curriculum.
